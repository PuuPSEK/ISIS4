Быстрая помощь для пожилых людей

«СоцПомощь» — сайт, где пожилой человек может оставить SOS-заявку, волонтёр — принять её в работу, а администратор — контролировать выполнение.

🚨 Подать SOS-заявку 📋 Посмотреть заявки

Как это работает

1. Пенсионер заполняет простую форму заявки.

2. Волонтёр видит заявку и принимает её в работу.

3. После оказания помощи статус заявки меняется на «Помощь оказана».

👵 Для пенсионеров

Можно быстро описать проблему, указать адрес и телефон для связи.

🧑‍⚕️ Для волонтёров

Доступен список заявок, которые можно принять и выполнить.

🛠 Для администратора

Администратор управляет статусами заявок и следит за порядком.
