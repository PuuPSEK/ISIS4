СоцПомощь (SOS Platform)

Веб-приложение для экстренной социальной помощи пожилым людям с ролевой системой.

Роли
👵 Пенсионер — создаёт SOS-заявки
🧑‍⚕️ Волонтёр — принимает заявки и помогает
🛠 Администратор — управляет заявками и статусами

Функционал
Создание SOS-заявок
Просмотр всех заявок
Обработка заявок волонтёрами
Управление статусами администратором
Фильтрация заявок
Ролевая система доступа
Хранение данных в localStorage
Технологии
HTML5
CSS3
JavaScript (Vanilla JS)
Node.js
Express.js

Установка и запуск

1. Установить зависимости
npm install
или (если нет package.json):
npm install express

2. Запуск сервера
node server/server.js

3. Открыть в браузере
http://localhost:3000

Структура проекта
public/
 ├── index.html
 ├── login.html
 ├── app.html
 ├── history.html
 └── assets/
 server/
 └── server.js

Важно
Сервер используется только для запуска сайта локально
Данные хранятся в localStorage
Перезапуск браузера может сбросить тестовые данные
