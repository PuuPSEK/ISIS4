# СоцПомощь (SOS Platform)

Веб-приложение для экстренной социальной помощи пожилым людям.

## Описание

«СоцПомощь» помогает пожилым людям быстро оставить заявку на помощь, а волонтёрам — увидеть обращение и принять его в работу.

## Роли

- Пенсионер — создаёт SOS-заявки
- Волонтёр — принимает заявки и отмечает выполнение
- Администратор — управляет всеми заявками и статусами

## Функционал

- вход по роли
- создание SOS-заявки
- просмотр списка заявок
- фильтрация по статусу
- поиск по заявкам
- изменение статуса заявки
- удаление заявки администратором
- тестовые записи для проверки работы

## Технологии

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- JSON-файл как простое внутреннее хранилище
- Render для публикации

## Запуск локально

```bash

```

http://localhost:3000
