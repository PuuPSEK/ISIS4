# СоцПомощь — внутренняя CRM

Веб-приложение для экстренной социальной помощи пожилым людям. Проект содержит собственную мини-CRM для обработки SOS-заявок без подключения внешних CRM-сервисов.

## Демо

```text
https://socpomosh.onrender.com
```

## Описание

«СоцПомощь» позволяет пенсионеру создать SOS-заявку, волонтёру принять её в работу, а администратору управлять заявками через внутреннюю CRM-панель.

В проекте используется демо-режим входа по ролям. Это сделано для учебной демонстрации логики прав доступа.

## Роли

- Пенсионер — создаёт SOS-заявки.
- Волонтёр — принимает заявки и завершает помощь.
- Администратор — работает с CRM-панелью, меняет статусы, приоритеты и комментарии.

## Функционал

- создание SOS-заявки;
- просмотр заявок;
- фильтрация по статусам;
- поиск;
- управление статусами;
- приоритеты заявок;
- назначение волонтёра;
- комментарий менеджера;
- статистика заявок;
- экспорт заявок в CSV;
- внутренняя CRM-панель.

## API

```text
GET    /api/health
GET    /api/sos
GET    /api/sos/stats
POST   /api/sos
PATCH  /api/sos/:id
DELETE /api/sos/:id
GET    /api/crm/export
```

## Этап 4 — собственная CRM-система

Так как настройка внешней CRM оказалась ограниченной, в проекте была развёрнута собственная внутренняя CRM. Она реализует основные функции CRM: хранение заявок, статусы обработки, приоритеты, назначение ответственного, комментарии менеджера, экспорт и статистику.
